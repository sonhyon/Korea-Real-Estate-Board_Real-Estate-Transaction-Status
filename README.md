# 아파트 매매 거래 데이터 분석 프로세스

## 1. 데이터 불러오기
- 현재 작업 디렉터리 확인 및 변경  
  - `getwd()`, `setwd()`
- 파일 불러오기  
  - `read.csv()`

## 2. 데이터 전처리
- 데이터 구조 확인  
  - `str()`, `summary()`
- 월 변수 형식 변환  
  - `"202501"` → `"2025-01-01"`  
  - `as.Date()`
- IQR 방식으로 이상치 판별 및 제거  
  - `quantile()`, `IQR()`
