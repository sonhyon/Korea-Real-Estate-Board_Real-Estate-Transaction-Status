<details>
<summary><strong>🛠 데이터 분석 전 준비</strong></summary>

<br>

① 패키지 로드 및 작업 경로 설정  

| 함수 | 설명 |
|---|---|
| `getwd()` | 현재 작업 디렉토리를 확인 |
| `setwd()` | 작업 디렉토리를 데이터 파일 경로로 변경 |

② 데이터 불러오기  

| 함수 | 설명 |
|---|---|
| `read.csv()` | CSV 파일을 데이터프레임 형태로 읽어옴 |
| `encoding` | 한글 깨짐 방지를 위한 문자 인코딩 지정 |

③ 데이터 구조 및 기초 정보 확인 (EDA 1단계)  

| 함수 | 설명 |
|---|---|
| `tail()` | 데이터 하위 일부 행을 확인 |
| `View()` | 스프레드시트 형태로 전체 데이터 확인 |
| `str()` | 변수 타입, 데이터 구조, 관측치 개수 확인 |
| `summary()` | 최소값, 최대값, 사분위수 등 기초 통계량 요약 |

④ 결측치 확인  

| 함수 | 설명 |
|---|---|
| `is.na()` | 각 값이 결측치인지 여부를 TRUE/FALSE로 판별 |
| `table()` | 결측치와 비결측치의 개수를 집계 |

⑤ 이상값 탐색 (EDA 2단계)  

| 함수 | 설명 |
|---|---|
| `boxplot()` | 사분위수 기반으로 데이터 분포와 이상값을 시각적으로 확인 |

⑥ 이상값 기준 확인 및 처리  

| 함수 | 설명 |
|---|---|
| `boxplot()$stats` | 박스플롯에서 최소값, 사분위수, 최대값 등 5수치 추출 |
| `ifelse()` | 조건을 만족하는 경우 값을 대체 |

⑦ 결측치 제거 후 분석용 데이터 생성  

| 함수 | 설명 |
|---|---|
| `filter()` | 조건에 맞는 행만 선택 |
| `!is.na()` | 결측치가 아닌 값만 추출 |

## 연도별·월별 거래량 증가 / 감소율

### ① 연도별 거래량 증감률 분석을 위한 데이터 생성

| 함수 | 설명 |
|---|---|
| `arrange()` | 시군구별 연도 순으로 정렬 |
| `mutate()` | 거래량 증감률 변수 생성 |
| `lag()` | 전년도 거래량 값 참조 |
| `ungroup()` | 그룹 해제 |


### ② 시군구별 거래량 증감률 시각화

| 함수 | 설명 |
|---|---|
| `filter(!is.na())` | 증감률이 계산되지 않은 첫 해 데이터 제거 |
| `geom_hline()` | 기준선(y = 0) 추가 |
| `geom_line()` | 연도별 증감률 추이 선 그래프 |
| `geom_point()` | 각 연도의 증감률 값 표시 |
| `facet_wrap()` | 시군구별 그래프 분할 |
| `theme_minimal()` | 최소한의 시각적 테마 적용 |

---

## 수도권 vs 비수도권 거래 동향 비교

### ① 데이터 생성

| 함수 | 설명 |
|---|---|
| `mutate()` | 기존 데이터에 새로운 변수(`권역`)를 생성 |
| `ifelse()` | 조건에 따라 값을 분기하여 수도권 / 비수도권 구분 |
| `%in%` | 특정 값이 지정한 집합에 포함되는지 여부를 판단 |
| `.groups = "drop"` | 요약 후 그룹 속성을 제거하여 일반 데이터프레임으로 반환 |
| `arrange()` | 권역 및 연도 기준으로 데이터 정렬 |

### ② 시각화
| 함수 | 설명 |
|---|---|
| `ggplot()` | ggplot2 기반 시각화의 기본 틀 생성 |
| `aes()` | x축, y축, 색상 등 시각적 요소를 데이터 변수에 매핑 |
| `geom_line()` | 연도별 거래량 추이를 선 그래프로 표현 |
| `geom_point()` | 각 연도의 거래량 값을 점으로 표시 |
| `labs()` | 그래프 제목 및 축 라벨 설정 |
| `theme_minimal()` | 불필요한 요소를 제거한 미니멀 테마 적용 |

💼 비즈니스적 의미  
- 유망 시장 / 침체 시장 구분  
- 투자 우선순위 지역 선정  
- 마케팅 집중 지역 결정
